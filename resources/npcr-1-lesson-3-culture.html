家庭成员和亲属的称谓

汉语亲属称谓不仅说明亲属关系，还常常区分父系与母系，以及同辈之间的年龄大小。

两个基本原则

一、区分父系和母系

父亲的父母和母亲的父母使用不同称谓。

二、区分年龄大小

同辈亲属常根据比自己年长还是年幼来称呼。

祖父母：父系与母系

关系常用称谓西班牙语
父亲的父亲祖父 / 爷爷 zǔfù / yéyeabuelo paterno
父亲的母亲祖母 / 奶奶 zǔmǔ / nǎinaiabuela paterna
母亲的父亲外祖父 / 外公 / 姥爷 wàizǔfù / wàigōng / lǎoyeabuelo materno
母亲的母亲外祖母 / 外婆 / 姥姥 wàizǔmǔ / wàipó / lǎolaoabuela materna

不同地区和家庭可能采用不同的口语称呼，例如母系祖父母可以称“外公、外婆”，也可以称“姥爷、姥姥”。学习时先掌握亲属关系和父系／母系的区别。
